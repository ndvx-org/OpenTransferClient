# OpenTransferClient
OpenTransferClient is a Application that provides a graphical user interface (GUI) for transferring messages and files between devices in a network.

It is designed to be user-friendly and efficient, allowing users to easily send and receive data without the need for complex command-line operations.

## Development status
OpenTransferClient is currently in early development stages. If you would like to contribute or provide feedback, please feel free to reach out ([GitHub Issues](https://github.com/ndvx-org/opentransferclient/issues), [GitHub Discussions](https://github.com/ndvx-org/opentransferclient/discussions), [Email](mailto:paw.all.obo.alexaro@gmail.com)).

## Language
OpenTransferClient is written in C++ and uses the SDL3 library for its graphical user interface.
